# موقع براند كروشيه ومكراميه

موقع إلكتروني عصري وجميل لمتجر تجار الكروشيه والمكراميه مع واجهة سهلة الاستخدام والتخصيص.

## 📁 محتويات الملفات

- **index.html** - الصفحة الترحيبية (صفحة البداية)
- **main.html** - الصفحة الرئيسية للمتجر مع قائمة المنتجات
- **product-details.html** - صفحة تفاصيل المنتج
- **styles.css** - ملف التنسيق الكامل (الألوان والتصميم)
- **script.js** - ملف JavaScript للوظائف والتفاعل

## 🎨 الألوان والتصميم

الموقع يستخدم:
- **الوردي الأساسي**: #FF69B4
- **الأبيض**: #FFFFFF
- **رموز زهور وديكور جميل**: 🌹 🌸 🌺 🌷
- **تصميم عصري وأنيق**

## 🚀 كيفية الاستخدام

### 1. فتح الموقع
- افتح ملف `index.html` في متصفح الويب
- ستظهر الصفحة الترحيبية مع اللوجو والاسم والزر

### 2. الانتقال للمتجر
- اضغط على زر "GO SHOPPING"
- سيتم الانتقال للصفحة الرئيسية

## 📝 إضافة المنتجات والأصناف

### إضافة منتج جديد:

**خطوة 1:** افتح ملف `script.js`

**خطوة 2:** ابحث عن قسم `const productsData =` في بداية الملف

**خطوة 3:** أضف منتج جديد بهذا الشكل:


## 🏷️ الأصناف المتاحة

الأصناف الرئيسية:
- **crochet** - كروشيه
- **macrame** - مكراميه
- **accessories** - اكسسوارات

يمكنك تغيير أو إضافة أصناف أخرى بتعديل الأزرار في `main.html` والبيانات في `script.js`

## 📞 تخصيص معلومات التواصل

افتح ملف `script.js` وابحث عن:

```javascript
const contactInfo = {
    whatsappNumber: "966501234567", // ضع رقم الواتساب الخاص بك (بدون +)
    phone: "+966 50 123 4567",      // رقم الهاتف
    email: "info@brandname.com",     // البريد الإلكتروني
    instagram: "https://instagram.com/brandname",
    facebook: "https://facebook.com/brandname",
    tiktok: "https://tiktok.com/@brandname"
};
```

استبدل هذه البيانات برقام وروابط التواصل الخاصة بك.

## 🖼️ إضافة صور المنتجات

يمكنك استخدام:
1. **روابط صور من الإنترنت** - انسخ الرابط المباشر للصورة
2. **صور محلية** - ضع الصور في نفس مجلد الموقع واستخدم المسار النسبي

### مثال:
```javascript
images: [
    "images/product1.jpg",
    "images/product2.jpg",
    "https://example.com/image.jpg"
]
```

## 🔧 نصائح للتخصيص

1. **تغيير الألوان**: عدّل قيم الألوان في ملف `styles.css` (متغيرات `--primary-pink`, `--light-pink` وغيرها)

2. **تغيير الخطوط**: عدّل خطوط Google في رؤوس ملفات HTML

3. **إضافة ميزات جديدة**: استخدم JavaScript في ملف `script.js`

4. **تغيير النصوص**: استبدل النصوص مباشرة في ملفات HTML
